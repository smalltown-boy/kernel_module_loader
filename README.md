# Kernel Module Loader

A GUI utility for loading and unloading Linux kernel modules.

## Description

Kernel Module Loader is a graphical user interface (GUI) application that simplifies the management of Linux kernel modules. With it, you can:

* load kernel modules;
* unload kernel modules;
* view a list of available and loaded modules;
* get information about the status of modules.

## Requirements

Before installation, make sure your system meets the following requirements:

* **OS:** Ubuntu 22.04;
* **Python:** version 3.13.3;
* **PyQt:** version 6.8.1.

## Launching the application

* Clone the repository;
* In the program directory, open a terminal and run the command `sudo python3 loader.py` (since the program loads and unloads kernel modules, it requires elevated privileges).
