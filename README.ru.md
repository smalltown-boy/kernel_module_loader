# Kernel Module Loader

Утилита с графическим интерфейсом для загрузки и выгрузки модулей ядра Linux.

## Описание

Kernel Module Loader — это приложение с графическим интерфейсом (GUI), которое упрощает управление модулями ядра Linux. С его помощью вы можете:

* загружать модули ядра;
* выгружать модули ядра;
* просматривать список доступных и загруженных модулей;
* получать информацию о статусе модулей.

## Требования

Перед установкой убедитесь, что ваша система соответствует следующим требованиям:

* **ОС:** Ubuntu 22.04;
* **Python:** версия 3.13.3
* **PyQt:** версия 6.8.1

## Запуск

* Клонируйте репозиторий;
* В директории программы запустите терминал и выполните команду `sudo python3 loader.py` (так как программа
осуществляет загрузку и выгрузку модулей ядра, ей требуются повышенные права)
